## Telespot ecosystem

Phone -> OSINT -> opt-out workflow:

```text
Telespot (CLI) -> Telespotter (Rust) -> TelespotXX (Web)
                          |
                          +-> xTELENUMSINT (Chrome extension)
                          +-> TeleSTOP (Desktop opt-out)
```
